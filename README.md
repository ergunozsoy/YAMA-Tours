# YAMA Kültür v2.0

Akademik kaynaklarla desteklenen, üç dilli kültür rotaları ve zamanda yolculuk platformu.

## Dosyalar
- `index.html` ana sayfa
- `style.css` tasarım
- `script.js` dil sistemi ve dinamik kartlar

## GitHub Pages
1. Dosyaları repo kök dizinine yükleyin.
2. Settings → Pages → Deploy from branch → main / root seçin.
3. Yayın linkini bekleyin.

## İçerik ilkesi
YAMA Kültür rastgele AI içeriği üretmez. Her önemli bilgi akademik literatür, DergiPark, tezler, müze/UNESCO belgeleri, seyahatnameler ve birincil kaynaklarla desteklenmelidir.
